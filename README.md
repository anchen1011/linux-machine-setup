# linux-machine-setup
Quick linux workstation setup **beta**

Performance is not stable yet.

Usage: 
```
$ chmod +x *.sh
$ ./setup.sh
[optional] $ isolate
[optional] $ remote
```
**Use isolate if you're not sure what you're doing.**
