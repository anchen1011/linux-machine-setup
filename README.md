# linux-machine-setup
Quick linux workstation setup **beta**

Performance is not stable yet.

Usage: 

For local machine: **use remote if you're not sure what you're doing.**
```
$ chmod +x *.sh
$ ./setup.sh
[optional] $ isolate
[optional] $ remote
```

For remote machine:
```
$ chmod +x *.sh
$ ./remote.sh
```
