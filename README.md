# linux-machine-setup
Quick linux workstation setup **beta**

Performance is not stable yet.

Usage: 
```
$ chmod +x *.sh
$ ./setup.sh
[optional] $ ./safe_mode.sh
[optional] $ ./devel_mode.sh
```
**Use safe mode if you're not sure what you're doing.**
